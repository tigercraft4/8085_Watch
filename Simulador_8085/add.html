#ADD

O Comando ADD somará o conteúdo de um registrador ou de M ao acumulador. Na soma o bit de carry (vai um) não será considerado. O resultado fica armazenado no acumulador. Seus formatos possíveis são :

 

ADD R

ADD M

 

Exemplo :

 

ADC C

 

Acumulador = 2Fh = 0010 1111b

Registrador B = 18h = 0001 1000b +

--- ----------

Soma 47h = 0100 0111b

 

Flags Afetados :

Z, S, P, C, AC

 

 

# O Flag Z (Zero) é setado sempre que o resultado de uma operação aritmética ou lógica é zero. Quando o resultado de uma conta não é zero, este Flag é zerado.
