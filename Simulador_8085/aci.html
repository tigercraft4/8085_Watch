#ACI

O Comando ACI somará um dado imediato ao acumulador, considerando na soma o bit de carry (vai um). Seu formato é :

 

ACI dado

 

Exemplo :

 

ACI 36h

 

Acumulador = 11h = 0001 0001b

Dado = 36h = 0011 0110b +

Carry bit = 01h = 0000 0001b +

--- ----------

Soma 48h = 0100 1000b

 

Flags Afetados :

ZFlagZ, S, P, C, AC
